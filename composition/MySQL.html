MySQL 社区版安装

macOS

网上有网文说, macOS的设计使得很多任务很容易做到, 但是MySQL安装却不是这样. Oracle官网针对macOS提供了两种安装方式, 一种是标准macOS安装包Native DMG, 另一种是通用二进制包. 当然Oracle官网的从源代码安装也应该是适用macOS, 这是第三种方法, 一般都是高级玩家用的. 当然, 还有第四种, 使用Homebrew, 那还必须提一下使用MacPorts安装 (这算第五种), 最后一种, 使用MAMP / XAMPP. 以下是列表总结:

  1. 标准macOS安装包Native DMG
  2. 通用二进制包
  3. 从源代码安装
  4. Homebrew
  5. MacPorts
  6. MAMP / XAMPP

还有虚拟机, 虚拟机管理, 容器三种方法:

  1. Virtual Box, VMware
  2. Vagrant
  3. Docker

当然, 选择Oracle提供的标准macOS安装包Native DMG应该最方便的, 使用System Preferences pane的方式来启动/暂停MySQL服务器 (可选系统启动时自动运行MySQL服务器).
